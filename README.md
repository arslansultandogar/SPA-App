Professional, minimal Next.js e-commerce starter showcasing product listing, filters, search, and a small dashboard.

## Overview

- **Stack:** Next.js (App Router), React, Tailwind CSS (configured), Node.js
- **Purpose:** Demo/storefront template with product fetch API, client-side filtering, pagination, and a dashboard area.

## Features

- Product listing with search, filters, and pagination
- Lightweight product API route (`/api/products`) for local/dev data
- Dashboard area for managing products
- Responsive components and simple UI primitives

## Quick Start

Prerequisites: Node.js 18+ and npm or pnpm

Install dependencies:

```bash
npm install
# or
pnpm install
```

Run development server:

```bash
npm run dev
# or
pnpm dev
```
